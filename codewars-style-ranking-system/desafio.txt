Crie uma classe chamada User que seja usada para calcular o quanto um usuário progredirá em um sistema de classificação semelhante ao usado no Codewars.

Regras de negócio:
Um usuário começa na classificação -8 e pode progredir até a classificação 8.
Não existe classificação 0 (zero). A próxima classificação depois de -1 é 1.
Os usuários irão completar atividades. Essas atividades também possuem classificações.
Cada vez que o usuário conclui uma atividade classificada, seu progresso de classificação é atualizado com base na classificação da atividade.
O progresso obtido com a atividade concluída é relativo à classificação atual do usuário em comparação com a classificação da atividade.
O progresso de um usuário começa em zero e, a cada 100 pontos, seu nível é elevado para o próximo.
Qualquer progresso restante obtido na classificação anterior será aplicado ao progresso da próxima classificação (não descartamos nenhum progresso). A exceção ocorre se não houver mais nenhuma classificação para a qual progredir (após atingir a classificação 8, não há mais progressão).
Um usuário não pode progredir além do nível 8.
O único intervalo aceitável de valores de classificação é -8, -7, -6, -5, -4, -3, -2, -1, 1, 2, 3, 4, 5, 6, 7, 8. Qualquer outro valor deve gerar um erro.
O progresso é avaliado da seguinte forma:

Concluir uma atividade com a mesma classificação que a do usuário valerá 3 pontos.
Concluir uma atividade que esteja classificada um nível abaixo da do usuário valerá 1 ponto.
Quaisquer atividades concluídas que estejam classificadas em 2 ou mais níveis abaixo da classificação do usuário serão ignoradas.
Concluir uma atividade com classificação superior à classificação atual do usuário acelerará sua progressão de classificação. Quanto maior a diferença entre as classificações, maior será o aumento na progressão. A fórmula é 10 * d * donde drepresenta a diferença de classificação entre a atividade e o usuário.
Exemplos de lógica:
Se um usuário com classificação -8 concluir uma atividade com classificação -7, ele receberá 10 pontos de progresso.
Se um usuário com classificação -8 concluir uma atividade com classificação -6, ele receberá 40 de progresso.
Se um usuário com classificação -8 concluir uma atividade com classificação -5, ele receberá 90 de progresso.
Se um usuário com classificação -8 concluir uma atividade de classificação -4, ele receberá 160 pontos de progresso, o que resultará em sua promoção para a classificação -7 e em um ganho de 60 pontos de progresso para a próxima classificação.
Se um usuário com classificação -1 concluir uma atividade de classificação 1, ele receberá 10 pontos de progresso (lembre-se, a classificação zero é ignorada).
Exemplos de uso do código:
var user = new User()
user.rank // => -8
user.progress // => 0
user.incProgress(-7)
user.progress // => 10
user.incProgress(-5) // will add 90 progress
user.progress # => 0 // progress is now zero
user.rank # => -7 // rank was upgraded to -7
Observação : O Codewars não utiliza mais esse algoritmo em seu sistema de classificação. Ele usa uma solução puramente matemática que fornece resultados consistentes, independentemente da ordem em que as atividades classificadas são concluídas.
