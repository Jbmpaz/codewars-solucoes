Um isograma é uma palavra que não possui letras repetidas, sejam elas consecutivas ou não. Implemente uma função que determine se uma string contendo apenas letras é um isograma. Considere que a string vazia é um isograma. Ignore maiúsculas e minúsculas.

Exemplo: (Entrada --> Saída)

"Dermatoglyphics" --> true
"aba" --> false
"moOse" --> false (ignore letter case)
