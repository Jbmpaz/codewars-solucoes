Neste kata, sua tarefa é criar todas as permutações de uma string de entrada não vazia e remover duplicatas, se houver.

Crie o máximo de "embaralhamentos" que conseguir!

Exemplos:

With input 'a':
Your function should return: ['a']

With input 'ab':
Your function should return ['ab', 'ba']

With input 'abc':
Your function should return ['abc','acb','bac','bca','cab','cba']

With input 'aabb':
Your function should return ['aabb', 'abab', 'abba', 'baab', 'baba', 'bbaa']
Observação: A ordem das permutações não importa.

Boa sorte!
